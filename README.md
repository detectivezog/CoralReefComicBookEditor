# Coral Reef Comic Book Editor
A basic comic book editor in html/java/css/csv

## How to Build
- **Splitting:** Select `Split V` or `Split H` and click a cell to divide it perfectly in half.
- **Adjusting:** Click and drag the invisible bars between cells. For exact ratios, click the bar to open the **Division Overlay**.
- **Cells:** Click a cell in `Edit Cell` mode to set gradients, padding, or background images via the sidebar.

## Doodles (Objects)
- Use the sidebar to add **Text**, **Images**, or **SVG Shapes** (Bubbles/Spiky/Clouds).
- Switch to `Doodle Tool` to move or resize objects using the blue handles.
- Click a doodle to open the **Doodle Overlay** to change shapes, font styles, or opacity.

## Saving & Exporting
- **Save CSV:** Downloads a manifest of your current project.
- **Load CSV:** Select a saved project file from your computer to resume work.
- **PNG Export:** Generates a high-resolution snapshot of your layout including margins.
